# 摘句抽屉

把偶然遇见的句子，收进一只私人抽屉。

## 功能

- 打开网页即可录入摘句，可补充出处和多个自定义标签
- 按时间排列，支持搜索内容、出处与标签
- 可点击的侧边月份轴
- 苹果系统字体／Noto Serif SC 宋体切换，朱红／松绿主题切换
- 标签创建、筛选、重命名与删除
- 编辑和删除摘录
- Markdown、JSON 备份导出与导入
- 本机保存、PWA 离线缓存与主屏幕安装

## 数据说明

摘录保存在当前浏览器的 `localStorage` 中，不会上传到仓库或服务器。不同设备不会自动同步，清除浏览器网站数据也会删除本地记录，请定期导出 JSON 备份。

首次联网打开网页后，页面文件会被缓存，之后可以离线打开使用。
